# Staś - Chan

## Instalacja
1. Sklonuj repozytorium (`git clone https://github.com/staszicwaw/staszicwaw.pl`)
2. Zainstaluj wszystkie biblioteki (`npm install`)
3. Zastosuj migracje na bazie danych (`npx prisma migrate dev`)

## Uruchamianie
### Środowisko dev
```shell
npm run dev
```

### Środowisko prod
```shell
npm run build
npm run serve
```
